Pixel Portrait & Thumbnail Concepts

This pass swaps the earlier vector approach for generated pixel graphics that feel closer to classic sim UI art and can double as sidebar portraits, table thumbnails, and event-card art. Everything below is rasterized in the browser and can be saved out as PNG.

Why pixel? Better fit for compact UI slots, stronger silhouettes at small sizes, and a more game-native feel than polished illustration mockups.
Coverage Aliens, planets, ships, systems, and event categories—basically the whole thumbnail ecosystem with fewer existential crises.
Workflow These are generated from code, so we can tune palettes, shapes, and faction style quickly without repainting every asset by hand.
Output Use the Save PNG button on any card to export a raster image for experiments, UI mockups, or placeholder asset packs.
Open via the dev server at /concepts/pixel-portraits/. The earlier SVG concepts are still available under /concepts/aliens/.
